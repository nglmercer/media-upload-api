# Getting Started

## Tech stack
- **Runtime**: Bun
- **Web framework**: Hono
- **Storage**: local filesystem for files, json-obj-manager (JSON file) for metadata

## Prerequisites
- [Bun](https://bun.sh/) installed on your machine.

## Quick start

1. **Install dependencies**
   ```sh
   bun install
   ```

2. **Run dev server (hot reload)**
   ```sh
   bun run dev
   ```

3. **Open in browser**
   ```
   http://localhost:3000
   ```

## Configuration
The application uses a configuration file `config.json` (created on first run) or environment variables.
